<h2>📖 Sobre o Projeto</h2>
<p>
  O <strong>Gerenciador de Tarefas</strong> é uma aplicação web desenvolvida para facilitar o controle de atividades diárias. Seja para uso pessoal ou para gestão de pequenas equipes, o sistema oferece uma interface limpa e responsiva. Ele permite que os usuários registrem, categorizem e acompanhem o progresso de suas demandas em tempo real, garantindo que nenhum prazo importante seja esquecido.
</p>

<h2>✨ Funcionalidades Principais</h2>
<ul>
  <li><strong>Gestão Completa (CRUD):</strong> Criação, visualização, edição e exclusão rápida de tarefas.</li>
  <li><strong>Sistema de Status:</strong> Acompanhe as tarefas movendo-as entre <em>Pendentes</em>, <em>Em Andamento</em> e <em>Concluídas</em>.</li>
  <li><strong>Priorização:</strong> Destaque atividades urgentes através de marcadores de prioridade (Alta, Média, Baixa).</li>
  <li><strong>Interface Responsiva:</strong> Acesso otimizado para desktops, tablets e smartphones (Mobile First).</li>
  <li><strong>Persistência de Dados:</strong> Armazenamento seguro e eficiente das informações de usuários e tarefas utilizando banco de dados relacional.</li>
</ul>

<h2>⚙️ Como Executar o Projeto Localmente</h2>
<p>Siga os passos abaixo para rodar a aplicação na sua máquina:</p>

<ol>
  <li><strong>Clone o repositório:</strong><br>
    <code>git clone https://github.com/seu-usuario/nome-do-repositorio.git</code>
  </li>
  <li><strong>Configure o Banco de Dados:</strong><br>
    Importe o arquivo <code>database.sql</code> (localizado na pasta raiz) para o seu servidor MySQL.
  </li>
  <li><strong>Conecte a Aplicação:</strong><br>
    Altere as credenciais de acesso ao banco de dados no arquivo de configuração do seu backend.
  </li>
  <li><strong>Execute:</strong><br>
    Inicie o seu servidor local (como XAMPP, WAMP ou Live Server) e abra a aplicação no navegador.
  </li>
</ol>
